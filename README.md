# MazeGame
A simple maze-solving game built with C# using OOP principles.

## Features
- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation

## How to Run
1. Clone the repository
2. Open MazeGame.sln in Visual Studio
3. Build and Run
